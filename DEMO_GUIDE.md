## рҹҺ¬ Complete User Journey

### Scenario: Student Finding Library

1. **Open App**
   ```
   User opens index.html
   вҶ’ Sees landing page
   вҶ’ Clicks "Launch Navigator"
   ```

2. **View Map**
   ```
   Map loads with all buildings
   вҶ’ 26 red pins visible
   вҶ’ Campus layout displayed
   вҶ’ Controls ready
   ```

3. **Select Start**
   ```
   User clicks "From" dropdown
   вҶ’ Selects "Main Gate"
   вҶ’ Red pin at Main Gate enlarges
   ```

4. **Select Destination**
   ```
   User clicks "To" dropdown
   вҶ’ Selects "Library Block"
   вҶ’ Green pin at Library appears
   ```

5. **Find Path**
   ```
   User clicks "Find Path" button
   вҶ’ Algorithm calculates route
   вҶ’ Blue line starts drawing
   вҶ’ Takes 1.5 seconds to complete
   ```

6. **Watch Animation**
   ```
   Route fully drawn
   вҶ’ Blue marker appears at start
   вҶ’ Marker moves along route
   вҶ’ Rotates to show direction
   вҶ’ Glows with pulsing effect
   вҶ’ Reaches destination
   вҶ’ Loops back to start
   ```

7. **Read Directions**
   ```
   Panel shows:
   вҶ’ Distance: 380 map units
   вҶ’ Step-by-step route
   вҶ’ All waypoints listed
   ```

8. **Control Animation**
   ```
   User clicks "Pause Animation"
   вҶ’ Marker stops moving
   вҶ’ Button changes to "Resume"
   вҶ’ User clicks "Resume"
   вҶ’ Animation continues
   ```

9. **Explore Map**
   ```
   User drags map to pan
   вҶ’ Scrolls to zoom in
   вҶ’ Clicks pins to see names
   вҶ’ Uses +/- buttons
   вҶ’ Clicks вҢӮ to reset
   ```

10. **Try Another Route**
    ```
    User clicks "Clear" button
    вҶ’ Route disappears
    вҶ’ Pins reset to red
    вҶ’ Ready for new search
    ```
