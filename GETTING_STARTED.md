# Getting Started with CIT Campus Navigator

## 🚀 Quick Start (30 seconds)

1. **Open the map:**
   - Double-click `index.html` 
   - Click "Launch Navigator"

2. **Find a route:**
   - Select "From" location (e.g., Main Gate)
   - Select "To" location (e.g., Library Block)
   - Click "Find Path"

3. **Watch the magic:**
   - Blue route draws automatically
   - Animated marker travels along the path
   - See distance and directions in the panel

**That's it! You're navigating! 🎉**

## 🎬 Animation Features

### What You'll See

1. **Route Drawing** (1.5 seconds)
   - Blue line draws from start to end
   - Smooth animation with glow effect
   - White dashes in the center

2. **Moving Marker** (continuous)
   - Blue dot travels along the route
   - Rotates to show direction
   - Pulsing glow effect
   - Loops automatically

3. **Visual Indicators**
   - Red pin = Start location
   - Green pin = End location
   - Blue arrow = Direction of travel
   - Blue pins = Locations on route

## 🔧 Troubleshooting

### Map not showing?
- Refresh the page (F5)
- Try a different browser
- Check if JavaScript is enabled

### Animation not smooth?
- Close other browser tabs
- Update your browser
- Try on a different device

### Can't select locations?
- Make sure you clicked "Find Path"
- Try using dropdowns instead of pins
- Clear route and try again

### Mobile issues?
- Use two fingers to zoom
- Make sure touch is enabled
- Try landscape orientation
