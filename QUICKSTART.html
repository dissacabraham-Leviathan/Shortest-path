Launch Campus Navigator →

Click the button above to open the interactive map

✨ Key Features

🎯 Smart Pathfinding
Uses Floyd-Warshall algorithm to find the shortest route between any two campus locations
🎬 Smooth Animations
Watch an animated marker travel along your route with Google Maps-style effects
📍 26 Campus Locations
All major buildings, hostels, sports facilities, and amenities mapped
🖱️ Interactive Controls
Pan, zoom, click pins, and control animations with intuitive controls

📖 How to Use

  1. Select Start Location
    • Use the "From" dropdown, OR
    • Click any red pin on the map
  2. Select Destination
    • Use the "To" dropdown, OR
    • Click another pin on the map
  3. Find Path
    • Click the "🗺 Find Path" button
    • Watch the blue route animate on the map
    • See distance and route details in the panel
  4. Control Animation
    • Use "⏸ Pause Animation" to stop the moving marker
    • Use "▶ Resume Animation" to continue
  5. Navigate the Map
    • Pan: Click and drag the map
    • Zoom: Scroll wheel or use +/- buttons
    • Reset: Click the ⌂ button to reset view

💡 Tips & Tricks

  • Mobile: Use pinch-to-zoom and drag gestures
  • Quick selection: Click pins directly instead of using dropdowns
  • Clear route: Use the "✕ Clear" button to start over
  • Watch animation: The blue marker shows real-time travel along the route
  • Route details: Check the panel for distance and step-by-step directions

🔧 Technical Details

Algorithm: Floyd-Warshall all-pairs shortest path (O(V³))

Nodes: 60+ road junction nodes for accurate routing

Technology: Pure HTML/CSS/JavaScript with SVG rendering

Animation: requestAnimationFrame for smooth 60fps animations
